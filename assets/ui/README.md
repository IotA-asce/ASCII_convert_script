# UI Assets

This folder contains optional UI artwork for the Streamlit app (`ascii_art/streamlit_gui.py`).

If present, the app will render:
- `assets/ui/hero.svg` (default, checked in)
- or `assets/ui/hero.png` / `assets/ui/hero.jpg` (if you replace it)

If you generate new artwork (for example with your preferred image model), keep the file name the same
so the app picks it up without code changes.

Suggested prompt (hero banner)
- Subject: "ASCII Converter" web app hero banner
- Style: retro terminal / halftone / scanline vibe, subtle, not too busy
- Composition: wide banner with empty center area for title
- Output: 1600x400 PNG
